# ClinicaVeterinaria - Sistema de Clínica Veterinária

Sistema web completo para gerenciamento de clientes, animais, serviços e consultas.  
Frontend: HTML/CSS/JS | Backend: Python Flask + SQLite  

Este README ensina como rodar o sistema **diretamente do terminal** em qualquer computador com Python 3.9+.

---

## 1. Pré-requisitos

- Python 3.9 ou superior
- Navegador moderno (Chrome, Edge, Firefox)
- Terminal ou Prompt de Comando

> Não é necessário instalar MySQL ou outro banco; o sistema usa SQLite embutido.

---

## 2. Rodando o sistema

1. Abra o terminal e navegue até a pasta do backend:

```bash
cd /caminho/para/Projeto Clinica Veterinaria/back-end

2. (Opcional) Ative um ambiente virtual:

```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

3. Inicie o servidor:

```bash
python app.py
```
